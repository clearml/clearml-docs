---
title: Platform Management Center
---

:::important
The Platform Management Center is available under the ClearML Enterprise plan.
:::

The Platform Management Center provides platform administrators with centralized control over all tenants across a 
ClearML Enterprise deployment. It enables platform administrators to monitor tenant activity, usage, and costs.

## Modules
* [Platform Overview](pmc_platform_overview.md) - Platform-wide visibility into compute resource utilization and metered usage and costs across all tenants.
* [Tenants](pmc_tenants.md) - Create and configure tenants: Set user quotas, generate admin credentials and configure tenant specific parameters. Each tenant's page provides a dashboard showing usage metrics and cost breakdowns, as well as information on that tenant's applications, volume templates, and template variable overrides 
* [Volume Templates](pmc_volumes.md) - Create and assign storage configuration templates to tenants, to configure storage for tenant workloads.
* [Template Variables](pmc_template_variables.md) - Define template variables for use in platform template specifications, with per-tenant override support.
* [Applications](pmc_apps.md) - Install, deploy, and manage ClearML applications across the deployment, with per-tenant availability control and launch form customization.
