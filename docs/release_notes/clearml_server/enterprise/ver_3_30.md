### Enterprise Server 3.30.0

**New Features and Improvements**

* ClearML Server
  * New Role-Based UI: The ClearML UI is now partitioned into role-specific experiences:
    * Tenant Admin - For managing the tenant: users, access controls, infrastructure, and tenant-wide activity.
    * AI Builder - For running and managing AI workloads: experiments, datasets, models, pipelines, and applications.
    * AI User - For running UI applications
  * Add support for user-provided credentials in third-party storage artifact cleanup on task deletion
  * Add Application category access rules, enabling admins to configure which user groups have access to which application category
  * Add task details to the Access Rules task selection dropdown to help distinguish between tasks with identical names
* Platform Management Center
  * Add UI tenant deletion and creation 
  * New Cross-Tenant Usage Dashboard: platform admins can view aggregated metered usage across all tenants
  * New Global Orchestration Dashboard: platform admins can monitor available and in-use compute resources across all tenants
  * Add Template Variables, allowing platform admins to define global template variables and tenant specific overrides
  * New look and feel: Full light/dark themes
  * New Applications Management: Platform admins can view installed apps, install new versions, assign apps to tenants, and remove apps

**Bug Fixes**
* Fix UI app instances archived during relaunch not being automatically deleted, along with their associated artifacts, after the relaunched instance completes
* Fix UI deletion of Service Account sometimes causes an impersonation error and for the user to be logged out
* Fix UI Resource Policies allows queue creation with a display name matching an existing queue
* Fix `Refresh` button incorrectly displaying in the UI Resource Configuration editor mode
* Fix Administrator Vault appearing empty and disabled after importing a file
* Fix Line Width slider length fluctuates when adjusted in UI task comparison scalar plots 
* Fix UI Settings > Application Gateway > Router page `Refresh` button sometimes doesn’t work
* Fix Platform Management Center side navigation menu sometimes disappears
* Fix UI Volume Templates displaying blank tenant entries for deleted tenants in the template list and Edit Template modal 
* Fix UI Applications page category header sometimes not displayed
* Fix user filter in UI object pages displaying selected users non-alphabetical order.
