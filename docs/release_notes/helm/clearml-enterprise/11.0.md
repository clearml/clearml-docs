### clearml-enterprise 11.0.0

**New Features and Improvements**
* Update ClearML Enterprise Server to [3.29](../../clearml_server/enterprise/ver_3_29.md) (`apiserver`/`fileserver` 3.29.1-800, 
  `webserver` 3.29.1-1888, `usage-aggregator` 3.29.0-25)
* Set server containers to run as non-root user by default
* Update `clearml-enterprise-agent-k8s-base` to 1.31-3.0.0-434 and `clearml-agent-services-app` to app-1.3.0-79
* Add option to configure `containerSecurityContext` for ClearML Applications via `clearmlApplications.basePodTemplate.containerSecurityContext`
