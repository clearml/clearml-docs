title: Version 10.12

### clearml-enterprise 10.12.6

**New Features and Improvements**
* Make fileserver reference optional for the Apps Agent

### clearml-enterprise 10.12.5

**New Features and Improvements**
* Skip generating `clearml-conf` credentials secret when `clearml.existingSecret` is set

**Bug Fixes**
* Fix custom `existingSecret` not being used in usage scripts

### clearml-enterprise 10.12.4

**New Features and Improvements**
* Add option to configure `livenessProbe`, `startupProbe`, and `readinessProbe` for apiserver, fileserver, and webserver
* Add option to configure `successThreshold` and `failureThreshold` for apiserver, fileserver, and webserver probes
* Set `periodSeconds` and `timeoutSeconds` default to `probeRoundtripSeconds` and `probeRoundtripTimeoutSeconds`

### clearml-enterprise 10.12.3

**New Features and Improvements**
* Update ClearML Enterprise Server to [v3.28.7](../../clearml_server/enterprise/ver_3_28.md#enterprise-server-3287)

### clearml-enterprise 10.12.2

**New Features and Improvements**
* Add label configuration support to Apps Agent pod template

### clearml-enterprise 10.12.1

**New Features and Improvements**
* Add toleration configuration support to Apps Agent pod template

### clearml-enterprise 10.12.00

**New Features and Improvements**
* Add HTTPRoute support
