### clearml-enterprise-agent 6.12.3

:::warning[Breaking Changes]
The agent image in chart v6.12.3 now runs as a non-root user. Depending on your configuration, some changes to your 
`values.yaml` may be required before upgrading. See [Upgrading to Agent Chart v6.12.3 or greater](../../../clearml_agent/enterprise_agent_upgrade_6_12_3.md) 
for details. 
:::

**New Features and Improvements**
* Set agent image to run as non-root user
